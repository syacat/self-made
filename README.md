# Personal Color Analysis (パーソナルカラー診断)

PHP自作

## 概要

パーソナルカラー診断サイトを作成しました。ログインユーザと一般ユーザに分け、それぞれでログインできるようにしています。

## 使い方

### ログインユーザ

診断結果の履歴を確認して削除できるように見れるようにしています。
また新しく診断すると履歴も更新されます

### 一般ユーザ

カラー診断を楽しめるようになっております。
